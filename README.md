# Personal Portfolio

This workspace contains a responsive personal portfolio site for Ngina Awan.

## Files

- `index.html` — homepage with portfolio sections
- `styles.css` — responsive styling
- `script.js` — mobile navigation and smooth scrolling
- `resource/Profile.pdf` — profile resume used for the site and download link

## Run locally

Open `index.html` directly in your browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000`.

## Deploy with GitHub Pages

1. Create a public repository named `misha-te.github.io`.
2. Upload the site files directly into the repository root. Do not add a containing folder.
3. In GitHub, go to `Settings → Pages`.
4. Select `main` branch and `/root` folder.
5. Save the settings.

The site should publish at: `https://misha-te.github.io/`

## Customize

- Update `index.html` to change the introduction, featured project details, experience, and contact links.
- Update `styles.css` to adjust colors, spacing, or layout.
- Keep `resource/Profile.pdf` in place if you want the resume download link to work.
